JsLocalSearch.js

JsLocalSearch es un buscador local, es decir busca dentro del html y es inmediato.
Sus funcionalida basica es buscar dentro listados no paginados ya sea resaltando u ocultando los que no coincidan con los criterios.
La filosofia de JsLocalSearch es la facilidad de implementacion sin olvidad la potencia , pero siempre buscando la simplicidad.

Paso 1 - Ejemplo Simple

Este es el ejemplo mas simple de todos, busca dentro de contenedores con la clase '.contsearch' en los hijos de clase '.gsearch' el texto de mas de tres silabas que se escribe en un input text con id "gsearchsimple"

Como funciona

Lo que no este dentro de .gsearch no sera evaluado. Su accion por defecto es ocultar los que no coinciden con la busqueda.
¿No entiendes lo que digo? te lo explico.
primero necesitas carga el jquery y el JsLocalSearch.js dentro de <head>:
<script src="https://code.jquery.com/jquery-2.2.4.min.js" ></script>
<script src="JsLocalSearch.js"></script>
 metemos en el body:
<div id="localSearchSimple"></div>
<script>$('#localSearchSimple').jsLocalSearch();</script>
Ahora necesitaremos un input de tipo texto para poder escribir las palabaras que fiultraran el resaultado, ponlo donde quieras dentro del body:
<input type="text" id="gsearchsimple">
Y ahora necesitamos crear la zona donde queremos buscar. En todos los sitios que queramos que se oculten si no coincide la busqueda le daremos la clase .contsearch, esto no quiere decir que busque en todo el contenedor, simplemente lo ocultara o no dependiendo de la busqueda:
<div class="contsearch simple" >
<div>Aqui no se busca porque no esta entro de .gsearch</div>
<div class='gsearch'>Aqui si que se busca porque esta dentro de .gsearch<
<div class='gsearch'>Aqui también que se busca porque esta dentro de .gsearch</div>
</div>
solo buscar dentro de los elementos marcados como .gsearch, es decir no buscara en todo el contendor.Un contebnedor puede tener varios .gsearch como area de busqueda.Puedes crear todos los contenedores que necesites.En el ejemplo de abajo hay cuatro.
De todas formas tienes un ejemplo simple , abrelo mira el codigo y seguro que te queda claro.

Como funciona

Cambiamos el constructor : 
     
        <script>
                $('#localSearchSimple').jsLocalSearch({
                "action": "Mark",
                "actionok": "mark", //es el estilo que se aplicara al contendor .contsearch si coincide
                "actionko": "unmark",//es el estilo que se aplicara  al contendor .contsearch si no coincide
            })
        </script>
Tendremos que definir los estilos: 
  
        <style >

            .mark {
                background-color: #d7ffe7 !important
            }
            .mark .gsearch{
                font-size: 20px
            }
            .unmark {
                background-color: #e8e8e8 !important
            }
            .unmark .gsearch{
                font-size: 15px
            }
        </style>

Quieres cambiar la clase de los contenedoires?

En un principio el input text es un id llamado '#gsearch', el contenedor es una clase llamada '.contsearch' y la clase donde busca es '.gsearch' pero las puedes cambiar muy facilmente, asi podras tener varios contenedores en una misma pagina!!! 
     <script>
        $('#localSearch5').jsLocalSearch(
            {
                "searchinput": "#gsearch_alternativo", //input text, puede ser una clase o un id
                "container": "contsearch_alternativo", //container , siempre una clase
                "containersearch": "gsearch_alternativo" //donde se busca m siempre una clase
        });
    </script>

Quieres cambiar el numero de caracteres para buscar?

    <script>
        $('#localSearch7').jsLocalSearch(
            {
                "mincaracteres": 1, //numero minimo de caracteres  para buscar, por defecto son 3
        });
    </script>
